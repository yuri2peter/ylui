## 部署

方案1和方案2任选其一。

#### 1. http-server（仅用于开发预览）

需要已安装Node.js+npm环境。

在项目目录输入命令：

```
npm i
npm start
```

根据提示访问对应网址。

#### 2. 借助web服务（部署至生产环境）

将代码部署至您的web服务目录，如nginx、apache服务器的www目录下，访问对应网址（如`http://127.0.0.1/ylui/`）。

> 能将web服务打包为独立应用的平台，如electron，Hbuilder云打包，也能正常工作。
> 切忌直接在文件管理器打开页面，将不能正常工作。
